# P5_01_shop 

# After installing npm (Node Package Manager), initialize it by executing the command 'npm init' in the terminal
# installing 'webpack', install Webpack CLI > npm install webpack webpack-cli --save-dev

# For compile the project with 'webpack', we have to execute the command <npm run build> in the terminal
# For stop npm, we have to execute the command <ctrl c> in the terminal

# Installing 'babel' a transpile in order to make our JavaScript code compatible with older browsers
# Install Babel CLI > npm install --save-dev babel-loader @babel/core @babel/preset-env babel-polyfill
 
# Installing a server to test your code and which automatically reloads your browser as soon as you change your code
# Install Webpack server CLI > npm install webpack-dev-server --save-dev

# Start the local server > npm start
# URL http://localhost:8080/
